Sobre mí

Experiencia laboral en: Manejo masivo de datos con Phyton, principalmente haciendo uso de la libreria Pandas. Aplicando POO (programacion orientada a objetos) en PHP y Laravel para el desarrollo de sitios web. Manejo de métricas empresariales y datos a gran escala con el lenguaje Phyton y Pandas. Gestión de base de datos en Mysql y BQ. Diseño e implementación de test para la completa seguridad de los proyectos. Conocimiento en herramientas como Google BigQuery, Cloud Functions, y todo el entorno del paquete Google Cloud. Uso de Git Flow y metodología Scrum para el trabajo en grupo sumado a herramientas de colaboración como GitHub y Bitbucket. Manejo de AWS específicamente en S3.
